模板文档管理

上传7类Word模板文档，支持后续系统填入数据并导出
模板占位符说明：
1. 基本信息占位符：
• {{NAME}} - 学生姓名
• {{STUDENT_NO}} - 学号
• {{TITLE}} - 论文/设计题目
• {{SUMMARY}} - 摘要（设计类型）
• {{YEAR}}、{{MONTH}}、{{DAY}} - 日期（年、月、日）

2. 成绩占位符：
• 论文类型：{{ITEM1}}、{{ITEM2}}、{{ITEM3}} - 各分项分数
• 论文类型指标名称：{{ITEM1_NAME}}、{{ITEM2_NAME}}、{{ITEM3_NAME}} - 评价指标名称
• 论文类型指标描述：{{ITEM1_DESC}}、{{ITEM2_DESC}}、{{ITEM3_DESC}} - 评价指标具体内容
• 论文类型满分：{{ITEM1_MAX}}、{{ITEM2_MAX}}、{{ITEM3_MAX}} - 各分项满分
• 设计类型：{{ITEM1}}~{{ITEM6}} - 各分项分数
• 设计类型指标名称：{{ITEM1_NAME}}~{{ITEM6_NAME}} - 评价指标名称
• 设计类型指标描述：{{ITEM1_DESC}}~{{ITEM6_DESC}} - 评价指标具体内容
• 设计类型满分：{{ITEM1_MAX}}~{{ITEM6_MAX}} - 各分项满分
• {{TOTAL}} - 总分（满分100）
注意：评价指标的名称、描述和满分值可在"评价指标管理"中配置
• {{COMMENT}} - 答辩评语
• 成绩评定表专用：{{ADVISOR_SCORE}}（指导教师评定成绩）、{{REVIEWER_SCORE}}（评阅人评定成绩）、{{DEFENSE_SCORE}}（答辩成绩）
• 成绩评定表加权：{{ADVISOR_30}}（指导教师30%）、{{REVIEWER_30}}（评阅人30%）、{{DEFENSE_40}}（答辩40%）、{{TOTAL_GRADE}}（总评成绩）

3. 签名占位符（图片）：
• {{SIGN_ADVISOR}} 或 {{SIGN_ADVISOR_TEACHER}} - 指导教师签名
• {{SIGN_REVIEWER}} 或 {{SIGN_REVIEWER_TEACHER}} - 评阅教师签名
• {{SIGN_LEADER}} 或 {{SIGN_GROUP_LEADER}} - 答辩组长签名
• {{SIGN_DEPT_HEAD}} 或 {{SIGN_DEAN}} - 系主任签名（成绩评定表专用）
• {{SIGN_JUDGE}}、{{SIGN_JUDGE_1}}、{{SIGN_JUDGE_2}}、{{SIGN_JUDGE_3}} - 评委签名（过程表，最多3个）

4. 小组统分表占位符（动态评委数量）：
• {{GROUP_NAME}} - 小组名称
• {{DEPT_NAME}} - 院系名称
• 学生行：{{STU_NAME_i}}（i为行号，从1开始）
• 评委列：{{JUDGE_j_SCORE_i}}（j为评委序号，i为学生行号）
• {{DEFENSE_SCORE_i}} - 答辩得分（第i个学生）
• {{FACTOR_i}} - 调节系数（第i个学生）
• {{FINAL_SCORE_i}} - 最后得分（第i个学生）
• 兼容旧格式：{{ROW_1_NAME}}、{{ROW_1_SCORES}}、{{ROW_1_AVG}}、{{ROW_1_FACTOR}}、{{ROW_1_FINAL}}等
